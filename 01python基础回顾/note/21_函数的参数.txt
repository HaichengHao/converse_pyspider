参数：可以在函数调用的时候给函数传递一些信息
分类:
    1.形参，在函数声明的时候需要准备一些变量来接收信息
        位置参数 args
        关键字参数 kwargs
        # 默认值参数:如果未指定则按默认值执行，如果指定则按指定的来执行
        注意，默认值参数应当放在非默认值参数之后，一般的规则是 func(args,kwargs,default argument)
    2.实参，实际在调用的时候传入的参数
    注意，在Python中的函数，一般都是(args,kwargs) 即位置参数在前，关键字参数在后
    正常情况下函数在执行的时候必须要保障形参有数据传入

    3.动态传参
        个数可变的位置参数 *args 返回的结果是一个元组
        个数可变的关键字参数 **kwargs 返回的结果是一个字典
        # 个数可变的位置参数和个数可变的关键字参数可以混合使用
        # 但是一定要注意顺序问题
        # 个数可变的位置参数需要放在个数可变的关键字参数之前
         Python 会按照既定的顺序以及规则(即位置参数返回的是一个元组，关键字参数返回的是一个列表)输出
         # 注意，一般将默认值参数放在位置参数之后，如果需要改变默认值则利用关键字传参即可
         # 总结位置顺序
         #  位置参数>可变位置参数>默认值参数>关键字参数>可变关键字参数
补充，如果想指定后面的是关键字参数的话只需(位置参数,*,关键字参数) 这样即可表示 *之后的参数是关键字参数
参数也可进行解包
对于位置参数的解包只需在要解包的对象前加上*,注意需要是列表或者元组
对于关键字参数的解包只需在要解包的对象前加上**,注意需要是字典
