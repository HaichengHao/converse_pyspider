条件判断在py中非常重要
if 语句的语法规则
   1.
    if 条件:
        执行语句

     含义：如果 条件成立 就 执行 执行语句
        如果不成立就不执行
   2.
    if 条件:
        执行语句1
    else:
        执行语句2

     含义：如果条件成立 就 执行语句1
            否则 执行语句2

   3.
    if 条件0:
        if 条件1:
            执行语句1
        else:
            执行语句2
    else:
        执行语句3
     含义：
        循环嵌套if
        如果满足条件0
            且满足条件1
           就执行语句1
           否则2
        否则
            执行语句3


   4.
     if 条件1:
        执行语句1
     elif 条件2：
        执行语句2
     ...
     else:
        语句0
      含义：如果条件1成立，则执行语句1，否则看条件2是否成立
      如果条件2成立则执行语句2，...如若都不成立，则执行语句0
