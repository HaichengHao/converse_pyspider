模板继承:
1. 多个模板具有完全相同的顶部和底部
2. 多个模板具有完全相同的模板内容,但是内容中部分不一样
3. 多个模板具有完全相同的模板内容

JINJA标签
{% block 名称%}
    base的信息
{% endblock %}

{% extends 'base.html'%}

{% block 名称%}
    要修改的信息
{% endblock %}


还有可以使用include导入自己想用的部分
{% include 'common/header.html'%}

使用场景:在A,B,C页面包含公共的部分,但是其他页面没有这部分
这个时候就考虑使用include




宏 macro
1. 把它看作是JINJA2 的一个函数,这个函数可以返回一个HTML字符串
2. 目的: 代码可以复用,避免代码冗余

{% macro 函数名(参数)%}
    函数体，一般是html标签
{% endmacro %}

调用
{{函数名}}
在一个文件里导入
{% import 'macro/macro.html' as f %}
{{f.函数名(参数) }}


设置全局变量和局部变量
{% set 全局变量名 = 值 %}
{{ 全局变量名 }}  调用

{% with 局部变量名 = 值 %}
    {{ 局部变量名 }}
{% endwith %}
